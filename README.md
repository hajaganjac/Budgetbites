# BudgetBites

Student-friendly budgeting and recipe web app that helps students plan cheap meals based on their weekly grocery budget.

## Features

- 🎮 **3D Animated Login** - Bun Bon the hamburger character with mouse tracking
- 💰 **Budget Planning** - Set your weekly grocery budget (€10-€80)
- 🍳 **Recipe Generator** - Smart recipes from Albert Heijn, Lidl, and Aldi
- 📊 **Calorie Tracking** - Automatic nutrition tracking
- 🏆 **Gamification** - Cooking streaks, XP, levels, and achievements
- 👤 **Profile Customization** - Name and emoji avatar

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS v4
- **Routing**: React Router v7
- **Animation**: Motion (Framer Motion)
- **Charts**: Recharts
- **UI Components**: Radix UI, Lucide Icons

## Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Deployment

This app is configured for deployment on Vercel:

1. Push to GitHub/GitLab/Bitbucket
2. Import project to Vercel
3. Deploy

See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed instructions.
